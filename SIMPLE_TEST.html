Test Server Connection Test with localhost

Instructions:

  1. First: Make sure server is running
    • Double-click START_SERVER.bat
    • OR run: python app.py
    • Wait for: Running on http://0.0.0.0:5000
  2. Then: Click the test buttons above
  3. If you see the login page → Server is working!
  4. If you see "Cannot reach" → Server is not running

Still Not Working?

1. Check server window - is it showing "Running on..."?

2. Make sure you're using http:// not https://

3. Try a different browser (Firefox or Edge)

4. Check Windows Firewall - allow Python on port 5000
